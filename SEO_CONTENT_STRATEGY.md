## Keyword Implementation Guidelines

### On-Page SEO Checklist for Each Page:

- [ ] **Title Tag**: Include primary keyword, keep under 60 characters
- [ ] **Meta Description**: Include primary + secondary keywords, 150-160 chars
- [ ] **H1**: Include primary keyword, only one H1 per page
- [ ] **H2s**: Include secondary keywords naturally
- [ ] **First Paragraph**: Include primary keyword in first 100 words
- [ ] **Keyword Density**: 1-2% (don't overstuff)
- [ ] **Internal Links**: At least 3-5 to related pages
- [ ] **External Links**: 1-2 to authoritative sources
- [ ] **Alt Text**: Describe images with keywords
- [ ] **URL**: Include primary keyword, keep short
- [ ] **Word Count**: Minimum 800 words for blog posts, 400+ for service pages

## Common SEO Mistakes to Avoid

❌ **Keyword Stuffing**
```diff
- "Best home tutor in Jodhpur, find home tutor Jodhpur, home tutor Jodhpur..."
+ "Find the best home tutor in Jodhpur who can help your child excel..."
```

❌ **Duplicate Content**
- Don't copy from other websites
- Make each page unique
- Use canonical tags if needed

❌ **Ignoring Mobile Users**
- Test on mobile devices
- Ensure fast loading
- Easy navigation

❌ **Slow Page Speed**
- Optimize images
- Minimize JavaScript
- Use caching

❌ **No Internal Linking**
- Link related pages together
- Use descriptive anchor text
- Create content silos
